# The Road Home — redesign prototype

A responsive, modern rebuild of [theroadhome.org](https://theroadhome.org/) — a 100-year-old homeless-services nonprofit in Salt Lake City. Built with the [stardust](https://github.com/excat/skills) design pipeline: brand → briefings → prototype → render.

**Open [`index.html`](./index.html) to browse.**

## Typography

Three fonts, no reliance on overused defaults:

- **Archivo** (Omnibus-Type) — display headings, 900 weight at hero scale
- **Public Sans** (USWDS) — body, the US-government civic sans
- **Source Serif 4** (Adobe) — editorial moments only (pull-quotes)

## Palette

## Caveats before ship

See `stardust/EXECUTION_LOG.md` for the full list. Highlights:

- Hero photography uses Unsplash placeholders in some slots — production requires real TRH photography
- Darnell's pull-quote on home / Palmer Court / Housing Stories is **fictional** — must be replaced with a real client story before production
- Team and Board rosters are partially synthesized — real staff from `/team-sitemap.xml` need to replace placeholder cards
- All forms are UI only — no real submission, no real payment integration
- Annual-report PDFs link to `#` placeholders — need hosted PDFs

## Platform

This is static HTML + CSS. No JS framework, no build step. Any downstream platform (AEM Edge Delivery, a Sanity + Next stack, Framer, Webflow, etc.) is a mechanical translation of the approved design into that platform's authoring model — out of scope for this prototype.
