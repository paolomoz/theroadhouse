The Road Home redesign

A rebuild of theroadhome.org on a modern, responsive design system — built through the stardust pipeline. Brand extracted from the live site, design personality captured from the designer, and the full set of 79 content pages rendered with verbatim copy from the live site (with targeted rewrites where the original was Lorem ipsum or failing). Twenty hero images generated with Gemini 3 Pro Image Preview; no photorealistic portraits of fabricated people.

The work below is a self-contained prototype. Every HTML file can be opened directly in a browser. Start with the home page, click through the nav, resize the window to see the responsive behavior.
